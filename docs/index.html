Quick Start

  1. Download the latest DMG from the releases page
  2. Mount the DMG and drag CTX to Applications
  3. Launch CTX from Applications folder
  4. Enjoy! Start exploring your Claude Code projects
Requirements:
  • macOS 10.13 or later
  • Claude Code CLI installed (pip install claude-code)
  • ~50MB free disk space

How It Works

CTX launches Claude Code sessions in the background and visualizes them in real-time. Each session appears as a node in the 3D visualization, with color indicating state (idle, working, waiting for input). Watch your multi-threaded workflows unfold with perfect clarity.
